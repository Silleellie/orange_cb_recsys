class orange_cb_recsys.recsys.algorithm.Algorithm¶

Bases: abc.ABC

Abstract class for an Algorithm.

Every algorithm must be able to predict or to rank, or maybe both. In case some algorithms can only do one of the two (eg. PageRank), simply implement both methods and raise the NotPredictionAlg or NotRankingAlg exception accordingly.

copy()¶

Make a deep copy the algorithm

abstract predict(**kwargs)¶

Method to call when score prediction must be done.

If the Algorithm can’t do score prediction, implement this method and raise the NotPredictionAlg exception

abstract rank(**kwargs)¶

Method to call when ranking must be done.

If the Algorithm can’t rank, implement this method and raise the NotRankingAlg exception

class orange_cb_recsys.recsys.methodology.AllItemsMethodology(items_list: Set[str])¶

Bases: orange_cb_recsys.recsys.methodology.Methodology

Module of the Evaluation pipeline which, given a ‘train set’ and a ‘test set’, has the task to calculate which items must be used in order to generate a recommendation list

With AllItemsMethodology, given a user U, items to recommend for U are all items that appear in ‘items_list’ parameter excluding those items that appear in the ‘train set’ of U

If the ‘only_greater_eq’ parameter is specified, then only items with rating score >= only_greater_eq will be returned

Parameters:

items_list (Set[str]) – Items set that must appear in the recommendation list of every user

    If True the output of the method will be a dict containing users as a key and list of item that must be predicted as a value. .. rubric:: Example

    {‘u1’: [‘i1’, ‘i2’, ‘i3’], ‘u2’: [‘i1’, ‘i4’], …}

Returns:

A DataFrame which contains all items which must be recommended to every user based on the methodology chosen.

Class container for two pandas DataFrame

It may represent a split containing ‘train set’ and ‘test set’, or a split containing a ground truth and predictions for it, etc.

Once instantiated, one can access the two dataframes in different ways:

> sp = Split()
> # Various ways of accessing the FIRST DataFrame
> sp.train
> sp.pred
> sp.first
>
> # Various ways of accessing the SECOND DataFrame
> sp.test
> sp.truth
> sp.second
Parameters:

  • first_set (pd.DatFrame) – the first DataFrame to contain. If not specified, an empty DataFrame with ‘from_id’, ‘to_id’, and ‘score’ column will be instantiated

  • second_set (pd.DataFrame) – the second DataFrame to contain. If not specified, an empty DataFrame with ‘from_id’, ‘to_id’ and ‘score’ column will be instantiated
